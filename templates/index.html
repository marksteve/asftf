What is Flask?

From the main website:

Flask is a microframework for Python based on Werkzeug, Jinja 2 and good intentions.

Installing Flask

# Python 2
pip install virtualenv
virtualenv asftf

# Python 3.3
venv asftf

source asftf/bin/activate
pip install Flask

Run examples on your own

git clone https://github.com/marksteve/asftf
cd asftf
# asftf virtualenv should be activated
python hello-world.py
